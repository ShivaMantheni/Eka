4 · Backend Flow — POST /api/vs/{dut_id}/spin

POST /api/vs/{dut_id}/spin
{ project, username, number, source_image }
↓
Build vs_name = project + username + number
xml_full = dut.xml_path + "/" + vs_name + ".xml"
img_dir = "/var/lib/libvirt/images/" + vs_name + "/"
↓
Step 1/5 · SSH: test -f {xml_full}
↓
Step 2/5 · SSH: sudo mkdir -p {img_dir}
↓
Step 3/5 · SSH: sudo cp {source_image} {img_dir}{filename}
↓
Step 4/5 · SSH: sudo virsh define {xml_full}
↓
Step 5/5 · SSH: sudo virsh start {vs_name}
↓
✓ VS Spun · Refresh VM list
